# MCPs

Canonical bucket for MCP registry and server configuration.

Current source: `global-opencode-config/mcp.json` and `.opencode/mcp.json`

Quickstart:
1. Keep the bucket as the visible registry home.
2. Mirror the local flat config only through the daemon.
3. Avoid direct edits to local copies unless you are debugging sync drift.
